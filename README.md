# 🩺 Diabetes Prediction using Logistic Regression

## 📌 Description

This project is a Machine Learning model that predicts whether a person is diabetic or not based on medical parameters. It uses the Logistic Regression algorithm to classify outcomes with good accuracy.

The model is trained on a dataset containing features like glucose level, blood pressure, BMI, age, etc.

---

## 🚀 Features

* Predicts diabetes (Positive / Negative)
* Data preprocessing and normalization
* Logistic Regression model implementation
* Model evaluation using accuracy and confusion matrix
* Clean and structured code

---

## 🛠️ Tech Stack

* Python 🐍
* NumPy
* Pandas
* Scikit-learn
* Matplotlib

---

## 📂 Dataset

The dataset contains the following features:

* Pregnancies
* Glucose
* Blood Pressure
* Skin Thickness
* Insulin
* BMI
* Diabetes Pedigree Function
* Age
